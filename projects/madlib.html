Madlib assignment

This was a our first javascript assignment that we did in partners. This javascript assignment made it so that we would use prompts to ask the user questions. These questions that we ask the user, we take their input and use their own input and relate it back to them in a sentence or another question.

Me and my partner decided to create a sort of restaurante with the prompts. We ask them what they want to order and what they want to drink. After that we ask how many of the items they want to add, and based on how many items we got the code to do math for us and then give them their total.
