Cyota Project

For this cyota project we had to collaborate with a partner on cs50 to create an adventure story which me and my partner (Katherine) named it "Trip to Space". We made our own adventure using files so when u click a file it will tell you what event happened and what you can do. When clicking these files it will send you into another file that will have an outcome to whatever you picked. In this project me and my partner made you a lucky person who gets to fly out to space for free by Nasa. You can choose different options on what to do but some of these options could get you killed or lead you to survival.

I think that our teacher assigned us this project to us because not only are we learing on how to use IDE with people but this project is also helping us practice our communication skills by talking with our partners and doing the work together. For me this project was not difficult at all since all we had to do was link files together and write a story inside each file that makes sense. One thing I would do differently if I had more time for this project I would have liked to add a bit more text and some images to make the files look more appealing to the person going through them.
